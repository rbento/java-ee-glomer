# java-glomer

A helper library for Java EE projects

###### About

- Targets MongoDB database with Morphia ORM library and Apache DeltaSpike / Security module

- Contains definitions that can be reused across similar ptojects, such as locale selection, custom annotation and annotation processing

###### Notice

- This project was developed back in 2012 so features related to APIs such as Morphia may be outdated
- Glomer should not mean anything, just a name
